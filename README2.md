# liugh-openResources
##开源后台管理系统
后端 Spring+SpringMVC+Mybatis
数据库 MySQL
前端 bootstrap


##有哪些功能？

* 系统基础管理
    *  用户管理
    *  角色管理
    *  菜单管理
* 系统实时监控
   *  告警列表
   *  实时监控
* 登录信息管理
   *  用户登录记录
* 操作日志管理
   *  日志查询
